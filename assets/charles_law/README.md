# Charles Law Activity SVG Assets

This directory should contain SVG files for the Balloon and Bottle experiment.

## Required SVG Files

Place the following SVG files in this directory:

- `cup.svg` - Cup/mug graphic
- `bottle.svg` - Transparent bottle graphic  
- `balloon.svg` - Balloon graphic
- `container.svg` - Container/beaker graphic
- `hot_water.svg` - Hot water icon/graphic
- `cold_water.svg` - Cold water icon/graphic

## Usage

The app will automatically use these SVG files if they exist. If an SVG file is missing, the app will fall back to Material Icons.

## SVG Requirements

- Vector format (SVG)
- Optimized for web/mobile
- Recommended size: 24x24 to 48x48 viewBox
- Should work well at different scales
